# Branding Assets

Place your logo and favicon files here:
- `logo.png` - Main logo image (recommended: 200x200 or higher)
- `favicon.ico` - Favicon (recommended: 32x32 or 64x64)

These files will be used across the site:
- Navbar logo
- Footer logo
- Email templates
- Browser favicon

The file paths can be customized via environment variables:
- `LOGO_URL` - Path to logo (default: /branding/logo.png)
- `FAVICON_URL` - Path to favicon (default: /branding/favicon.ico)
